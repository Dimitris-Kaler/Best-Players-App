# Best-Players-App

#Set Up 
1.Clone down this repository
`git clone https://github.com/Dimitris-Kaler/Best-Players-App`

2.Install depedencies
`npm i `
Make sure tha you are in the proper folder 

3.Start the server

`npm start`
